# Usage
<!-- usage -->
```sh-session
$ npm install -g appium-cli
$ appium COMMAND
running command...
$ appium (-v|--version|version)
appium-cli/0.0.0 darwin-x64 node-v10.11.0
$ appium --help [COMMAND]
USAGE
  $ appium COMMAND
...
```
